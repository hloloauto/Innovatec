
# App de Gestión de Tareas Universitarias

Proyecto desarrollado con enfoque Low-Code y tecnologías web.

## Tecnologías
- HTML
- CSS
- JavaScript
- Google Sheets (base de datos)
- Glide (prototipo móvil)

## Funcionalidades
- Registro de tareas
- Seguimiento de estado
- Visualización tipo dashboard
- Métricas de desempeño
